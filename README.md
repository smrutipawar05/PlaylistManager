# Playlist Manager
A command-line Playlist Manager built using Python and Object-Oriented Programming.

## Overview
Built to learn software engineering by developing a real application.
It allows users to manage a song library and playlists through a command-line interface.

## Features
- Add songs to the song library
- View all songs
- Create playlists
- View all playlists
- Delete playlists
- Add songs to playlists
- Remove songs from playlists
- View songs inside a playlist
